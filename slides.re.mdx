# VARIABLES

Una variable es como una caja en la que puedes guardar datos. Esta caja tiene un nombre que usamos para referirnos a los datos guardados en ella.

**Los nombres de las variables no pueden contener espacios.**

<SyntaxHighlighter language="javascript" style={dark}>
  {codeBlocks.variableExample}
</SyntaxHighlighter>

---

# ¿LOCAL O GLOBAL?

Si la variable está declarada dentro de una función esta sera de caracter **LOCAL** por lo cual solo se podra acceder al valor dentro de la propia función.

Una variable **GLOBAL** es declarada fuera y puede accederse desde otras funciones.

<SyntaxHighlighter language="javascript" style={dark}>
  {codeBlocks.varScope}
</SyntaxHighlighter>

# VARIABLE NO MODIFICABLE

Si queremos fijar una variable para que no pueda ser modificada la declararemos usando **const** (de constante) en lugar de var.

# CONDICIONALES

Se usan para que el código ejecute diferentes bloques basándose en si una condición es verdadera o falsa y nos permiten preparar la lógica para responda a diferentes situaciones de manera dinámica. Podemos usar los siguientes tipos:

- **if** (si): Si se cumple esta condición...
- **else if**: Lo usaremos para concatenar mas condiciones.
- **else** (si no): Si las condiciones anteriores no se cumplen...

<SyntaxHighlighter language="javascript" style={dark}>
  {codeBlocks.conditionalSyntaxt}
</SyntaxHighlighter>

---

### EJEMPLO

---

### OPERADORES PARA CONDICIONALES

- IGUAL (valor): **==**
- IGUAL (tipo y valor): **===**
- NO IGUAL: **!=** / **!==**
- MENOR QUE: **&lt;**
- MENOR O IGUAL QUE: **&lt;=**
- MAYOR QUE: **&gt;**
- MAYOR O IGUAL QUE: **&gt;=**
- Y: **&&** (Si se cumplen ambas)
- O: **||** (Si se cumplen alguna)
